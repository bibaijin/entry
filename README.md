## 打包上传到 PyPI

### 依赖

```
pip install twine  # 上传工具
pip install wheel  # 打包工具
```

### 打包上传

```
rm -rf dist/  # 清空以前的构建
python setup.py sdist  # 打包源代码
python setup.py bdist_wheel  # 构建 wheel
twine upload dist/*  # 上传
```
